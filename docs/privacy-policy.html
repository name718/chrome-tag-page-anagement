TabTamer 隐私权政策

v1.0 · Last updated: 2025-08-18

本隐私政策适用于 Chrome 扩展程序「TabTamer - 标签页管理大师」。我们非常重视您的隐私，仅在本地处理必要的数据以提供标签页分组、休眠与快照等功能。

我们收集哪些数据

为实现功能，扩展可能在本地处理以下信息：

  • 标签页元数据：页面标题（document.title）、网址（URL）、站点域名（hostname）
  • 标签页状态：是否休眠（discarded）、固定（pinned）、活动状态（active）、上次活跃时间等
  • 窗口及分组信息：用于本地分组与快照管理

上述数据仅用于在本地设备上计算与展示，不会上传至任何服务器。

数据如何使用

  • 对标签页进行分组（按域名/关键字/时间等策略）
  • 实现休眠/唤醒以优化内存使用
  • 在本地保存工作区快照，便于恢复

数据存储与传输

  • 全部数据保存在本地（Chrome storage.local），不会上传至开发者或第三方服务器
  • 扩展不包含任何远程数据采集 SDK，不进行广告追踪

与第三方共享

我们不出售、出租或与第三方共享您的任何个人信息。

用户控制与透明度

  • 您可在扩展的“选项”页面导出/导入或清除数据
  • 您可随时卸载扩展以删除本地数据

权限说明

  • tabs / windows：用于读取标签状态与窗口信息以完成分组/统计
  • storage：用于在本地保存设置、分组和快照
  • activeTab：在用户操作时访问当前活动标签
  • host_permissions: <all_urls> 与内容脚本：用于读取当前页面的标题/URL 以进行本地分组（不上传）

未成年人隐私

本扩展不面向未成年人，也不有意收集儿童的任何信息。

政策更新

我们可能不时更新本政策。变更将发布于本页面，重要变更会在版本更新说明中提示。

联系我们

若您对本政策有任何问题或建议，请通过 GitHub Issues 与我们联系： GitHub Issues

TabTamer Privacy Policy (English)

v1.0 · Last updated: 2025-08-18

This policy applies to the Chrome extension “TabTamer”. We respect your privacy. All processing is done locally to provide tab grouping, memory saving (discard), and workspace snapshots.

Data We Process

  • Tab metadata: page title (document.title), URL, domain (hostname)
  • Tab state: discarded, pinned, active, last active time
  • Window and group metadata for local grouping and snapshots

All data stays on your device and is not transmitted to any server.

How We Use Data

  • Group tabs (by domain/keyword/time/manual)
  • Discard/wake tabs to optimize memory usage
  • Create local workspace snapshots for recovery

Storage & Transfer

  • All data is stored locally via Chrome storage.local
  • No remote analytics, no ad tracking

Sharing

We do not sell, rent, or share your personal information with third parties.

User Control

  • Export/Import or clear data in the extension Options
  • Uninstall the extension at any time to remove local data

Permissions

  • tabs / windows: read tab/window state for grouping and stats
  • storage: persist settings, groups, and snapshots locally
  • activeTab: access the active tab upon user interaction
  • host_permissions: <all_urls> and content scripts: read page title/URL for local grouping (no transmission)

Children’s Privacy

Our extension is not directed to children and we do not knowingly collect information from children.

Changes

We may update this policy from time to time. Changes will be posted on this page.

Contact

Please open a ticket on GitHub Issues if you have questions.
